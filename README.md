# My Trading Algorithm Bot

This Trading Bot runs on [Alpaca's](https://alpaca.markets/) trading accounts.

## Features
- Once started, it will run until manually stopped or market hours are over.
- DualLogger: Logs to both console and file at the same time, so we can check logs in real-time and EOD review
- Supports multiple strategies
- Flexible configuration via command line arguments, for all accounts (paper or real) and strategies

## Strategies
- Pair Z-Scores
- MACD Crossover (Coming Soon)
